# React Course

Este repositorio contiene todos los materiales, ejercicios y proyectos desarrollados durante el curso de React. Está organizado en varias carpetas, cada una correspondiente a diferentes módulos, prácticas y ejercicios del curso.

## Estructura del repositorio

- **React-1/**: Introducción a React y fundamentos básicos.
  - Subcarpetas con proyectos y ejercicios de los primeros módulos (componentes, props, eventos, renderizado condicional, listas, etc.).
- **React-2/**: Profundización en props, comunicación entre componentes y renderizado avanzado.
  - Proyectos y ejemplos prácticos.
- **React-2-exercises/**: Ejercicios prácticos y soluciones de la segunda parte del curso, incluyendo el proyecto Tic-Tac-Toe.
- **React-3/**: Hooks, formularios, efectos y consumo de APIs.
  - Proyectos para practicar useEffect, formularios y fetch API.
- **React-3-exercices/**: Ejercicios y proyectos guiados para construir un carrito de compras paso a paso, con soluciones y recursos de apoyo.

## ¿Cómo usar este repositorio?

1. Navega por las carpetas según el tema o módulo que quieras repasar o practicar.
2. Lee los archivos README.md de cada carpeta para instrucciones y detalles específicos.
3. Ejecuta los proyectos localmente siguiendo las instrucciones de instalación de cada uno.

## Recomendaciones

- Realiza los ejercicios en orden para aprovechar el aprendizaje progresivo.
- Consulta las soluciones solo después de intentar resolver los ejercicios por tu cuenta.
- Usa la documentación oficial de React como referencia adicional.
