# React Basics - Proyectos Educativos

Colección de proyectos diseñados para aprender React desde cero, cubriendo conceptos fundamentales de manera progresiva.

## Proyectos Incluidos

### React-Basics-12-useEffect
**Tema**: Hook useEffect y ciclo de vida de componentes

Aprende a ejecutar efectos secundarios en componentes funcionales:
- useEffect sin dependencias, con array vacío y con dependencias
- Montaje y desmontaje de componentes
- Función de limpieza (cleanup)
- Renderizado condicional con Toggle

**Conceptos clave**: useEffect, ciclo de vida, montaje/desmontaje

---

### React-Basics-13-forms
**Tema**: Manejo de formularios en React

Aprende a capturar y procesar datos de usuario:
- Eventos onChange y onSubmit
- preventDefault para evitar recargas
- Actualización dinámica de campos
- Comunicación de hijo a padre mediante props
- Renderizado condicional basado en datos

**Conceptos clave**: formularios, eventos, onChange, onSubmit, props

---

### React-Basics-14-fetch-API
**Tema**: Consumo de APIs externas

Aprende a obtener y mostrar datos de APIs:
- Peticiones HTTP con fetch nativo y axios
- useEffect para llamadas a APIs
- Manejo de estados con datos remotos
- Renderizado de listas con map()
- Transformación de datos JSON

**Conceptos clave**: fetch API, axios, HTTP, APIs REST, useEffect con APIs

---

## Orden Recomendado

1. **useEffect** → Entender el ciclo de vida
2. **forms** → Capturar datos del usuario
3. **fetch-API** → Obtener datos externos

## Instalación

Cada proyecto se ejecuta independientemente:

```bash
cd React-Basics-XX-nombre
npm install
npm run dev
```

## Tecnologías

- React 18
- Vite
- JavaScript (ES6+)
- CSS3
