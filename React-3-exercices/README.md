
# React 3 Exercises

Este directorio contiene ejercicios prácticos para la tercera parte del curso de React, enfocados en la construcción de un carrito de compras paso a paso.

> **Referencia:** La carpeta `cart/` contiene una versión **JavaScript puro (vanilla)**, es decir, implementado sin frameworks, usando únicamente HTML, CSS y JavaScript estándar.

## Estructura de la carpeta

- **README-cart-01.md** a **README-cart-06.md**: Enunciados y guías para cada etapa del ejercicio del carrito.
- **cart/**: Carpeta auxiliar o de referencia.
- **cart-react-01/** a **cart-react-06/**: Proyectos de React que corresponden a cada etapa del desarrollo del carrito, desde la estructura básica hasta la integración de carga de datos con useEffect y Fetch API.
- **README-cart.md**: Resumen general del ejercicio del carrito.

## ¿Cómo usar este repositorio?

1. Lee los archivos `README-cart-01.md` a `README-cart-06.md` para entender cada etapa.
2. Trabaja en las carpetas `cart-react-01` a `cart-react-06` siguiendo el avance sugerido.
3. Consulta el archivo `README-cart.md` para una visión general del ejercicio.

## Recomendaciones

- Realiza cada etapa en orden para comprender la evolución del proyecto.
- No avances a la siguiente carpeta sin haber entendido la anterior.
- Consulta la documentación oficial de React si tienes dudas.

¡Éxito programando tu carrito en React!
