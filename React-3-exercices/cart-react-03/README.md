# Cart React 03 - Estado e Interactividad con useState

Este proyecto corresponde a la **tercera etapa** del tutorial Cart React.

## Documentación

Para seguir el tutorial completo, consulta: [README-cart-03.md](../README-cart-03.md)

## Conceptos de esta versión

- Hook useState para gestionar estado
- Manejo de eventos (onClick)
- Funciones como props (callbacks)
- Inmutabilidad en React
- Actualización dinámica del contador

## Instalación y ejecución

```bash
npm install
npm run dev
```

## Contenido

- Estado cartItems con useState
- Función handleAddToCart para agregar libros
- BookItem es clickeable
- El contador se actualiza dinámicamente
- Aún no se muestra el contenido del carrito
