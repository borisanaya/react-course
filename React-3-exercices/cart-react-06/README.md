# Cart React 06 - Carga de Datos con useEffect y Fetch

Este proyecto corresponde a la **sexta y última etapa** del tutorial Cart React.

## Documentación

Para seguir el tutorial completo, consulta: [README-cart-06.md](../README-cart-06.md)

## Conceptos de esta versión

- Hook useEffect para efectos secundarios
- Fetch API para cargar datos
- Promesas y async/await
- Carga de datos desde archivo JSON
- Separación de datos y código
- Estados de carga

## Instalación y ejecución

```bash
npm install
npm run dev
```

## Contenido

- Datos cargados desde public/books.json
- Hook useEffect con array de dependencias vacío
- Estado books gestionado con useState
- Fetch API para cargar datos
- Renderizado condicional para loading
- Aplicación completa y funcional
