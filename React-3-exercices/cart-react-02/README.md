# Cart React 02 - Componentes Reutilizables y Props

Este proyecto corresponde a la **segunda etapa** del tutorial Cart React.

## Documentación

Para seguir el tutorial completo, consulta: [README-cart-02.md](../README-cart-02.md)

## Conceptos de esta versión

- Props (propiedades)
- Desestructuración de props
- Componente BookItem reutilizable
- Flujo de datos padre → hijo
- Template literals

## Instalación y ejecución

```bash
npm install
npm run dev
```

## Contenido

- Header recibe cartCount como prop
- BookList recibe array de libros como prop
- Nuevo componente BookItem para cada libro
- Datos centralizados en App.jsx
