# Cart React 04 - Renderizado Dinámico con map()

Este proyecto corresponde a la **cuarta etapa** del tutorial Cart React.

## Documentación

Para seguir el tutorial completo, consulta: [README-cart-04.md](../README-cart-04.md)

## Conceptos de esta versión

- Método .map() para renderizar listas
- Prop key en listas de React
- Renderizado dinámico y escalable
- Arrow functions y retorno implícito
- Código más mantenible

## Instalación y ejecución

```bash
npm install
npm run dev
```

## Contenido

- Uso de .map() en BookList
- Prop key para identificar elementos
- 6 libros en lugar de 3
- Código que escala automáticamente
- Sin código repetitivo
