# Cart React 05 - Visualización y Gestión del Carrito

Este proyecto corresponde a la **quinta etapa** del tutorial Cart React.

## Documentación

Para seguir el tutorial completo, consulta: [README-cart-05.md](../README-cart-05.md)

## Conceptos de esta versión

- Componente Cart para mostrar el carrito
- Función para eliminar items (.filter())
- Renderizado condicional (&&)
- Toggle para mostrar/ocultar carrito
- Layout de dos paneles
- Estados booleanos

## Instalación y ejecución

```bash
npm install
npm run dev
```

## Contenido

- Nuevo componente Cart
- Estado showCart para toggle
- Función handleRemoveFromCart
- Función handleToggleCart
- Header clickeable para mostrar/ocultar carrito
- Panel lateral con items del carrito
