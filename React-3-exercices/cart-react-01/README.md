# Cart React 01 - Estructura Básica

Este proyecto corresponde a la **primera etapa** del tutorial Cart React.

## Documentación

Para seguir el tutorial completo, consulta: [README-cart-01.md](../README-cart-01.md)

## Conceptos de esta versión

- Componentes React básicos
- Estructura de carpetas
- Composición de componentes
- Renderizado estático (sin interactividad)

## Instalación y ejecución

```bash
npm install
npm run dev
```

## Contenido

- Header estático con contador fijo
- Lista de 3 libros hardcodeados
- Sin interactividad
- Datos escritos directamente en el JSX
