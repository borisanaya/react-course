# React 2 Exercises

Este directorio contiene ejercicios prácticos para reforzar los conceptos aprendidos en la segunda parte del curso de React.

## Estructura

- **README-01.md** a **README-06.md**: Enunciados de los ejercicios.
- **solutions/**: Soluciones propuestas para cada ejercicio, organizadas por carpeta.
- **Tic-Tac-Toe-initial/**: Proyecto base para el ejercicio de Tic-Tac-Toe.

## ¿Cómo usar este repositorio?

1. Lee los enunciados en los archivos `README-01.md` a `README-06.md`.
2. Realiza los ejercicios en tu propio entorno o usando el proyecto base si corresponde.
3. Consulta las soluciones en la carpeta `solutions/` solo después de intentar resolver los ejercicios por tu cuenta.

## Recomendaciones

- Intenta resolver cada ejercicio antes de ver la solución.
- Usa el proyecto base para practicar con código real.
- Si tienes dudas, revisa la documentación oficial de React o los apuntes del curso.

¡Mucho éxito programando en React!
